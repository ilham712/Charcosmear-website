## 📋 Tentang Proyek

**Charcosmear** adalah website landing page untuk produk perawatan motor ramah lingkungan yang terbuat dari bahan alami seperti charcoal, coconut oil, dan lemon oil. Website ini dirancang dengan tema dark & premium untuk target pasar Gen Z (15-24 tahun).

> *"Karena yang bagus itu bukan cuma tampilan, tapi juga impact-nya."* 🌱

## 🛠️ Cara Menjalankan

### Opsi 1: Langsung Buka File
1. Clone atau download repository ini
2. Buka file `index.html` di browser (Chrome/Firefox/Edge)
3. Done! Website langsung berjalan

### Opsi 2: Live Server (Recommended untuk Development)
1. Install extension **Live Server** di VS Code
2. Klik kanan pada `index.html`
3. Pilih **"Open with Live Server"**
4. Browser akan otomatis terbuka dengan hot reload

### Opsi 3: Deploy ke GitHub Pages
1. Push semua file ke repository GitHub
2. Buka **Settings** → **Pages**
3. Pilih branch `main` dan folder `/ (root)`
4. Klik **Save**
5. Website akan live di `https://username.github.io/repo-name`

## 🔧 Customization

### Mengganti Gambar
1. Siapkan gambar dengan format `.jpg` atau `.png`
2. Simpan di folder `assets/images/`
3. Update path di `index.html` atau `css/style.css`

### Mengubah Warna
Edit di file `index.html` bagian `<script>` Tailwind config:
```javascript
colors: {
  gold: '#D4AF37',        // Ganti kode warna di sini
  olive: '#6B7C3A',
  // ...
}
```
